## Review checklist for each release

Before tagging a new version, a maintainer should verify:

- [ ] All ACL Anthology links resolve.
- [ ] Page-limit and limitations-section requirements still match the current ACL CFP.
- [ ] Responsible NLP Checklist link is current.
- [ ] Best-papers chapter includes at least one paper from the most recent ACL.
- [ ] No paper cited as an example has been retracted.
- [ ] Date in the README "last reviewed" badge updated.
